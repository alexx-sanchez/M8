Bienvenidos a la Página Web de Prueba

Introducción

Esta es una página de prueba creada para demostrar cómo se puede construir una estructura básica de una página web usando HTML y CSS. El propósito de este sitio es ofrecer un ejemplo funcional y sencillo para aprender los fundamentos del desarrollo web.

Este texto es completamente inventado, pero representa una idea básica de lo que podrías incluir en una página de inicio para un sitio personal o un proyecto en desarrollo.
